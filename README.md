# MindMirror

MindMirror is a small playground for browser-based computer vision demos and experiments.
The focus is on quick, interactive prototypes you can open locally and learn from.

## What's inside

- `computer-vision/`: HTML/JS demos, utilities, and step-by-step guides.

## How to run

1. Open any HTML file in `computer-vision/` with your browser.
2. Follow the instructions in the corresponding guide files.

## Notes

Some demos may request camera access depending on the feature being tested.
